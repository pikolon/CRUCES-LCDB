Configurar Firebase
Para sincronizar pantalla, admin y control remoto en tiempo real necesitás conectar Firebase Realtime Database. Es gratuito y tarda unos 5 minutos.
1Entrá a console.firebase.google.com → Agregar proyecto → ponele un nombre (ej: fight-night).
2Compilación → Realtime Database → Crear base de datos → elegí región → Iniciar en modo de prueba.
3En la pantalla principal hacé clic en </> → Agregar app web → copiá el objeto firebaseConfig que aparece.
4Abrí fight-night.html con un editor de texto, buscá FIREBASE CONFIG y pegá los valores. Guardá y volvé a subir a GitHub Pages.
const FIREBASE_CONFIG = { apiKey: "AIzaSy...", authDomain: "fight-night.firebaseapp.com", databaseURL: "https://fight-night-default-rtdb.firebaseio.com", projectId: "fight-night", storageBucket: "fight-night.firebasestorage.app", messagingSenderId: "123456789", appId: "1:123456789:web:abc123" };
Sin Firebase la app funciona solo en el dispositivo actual (sin sincronización entre pantalla, admin y celular).
