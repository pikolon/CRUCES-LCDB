Configurar Firebase
Para sincronizar pantalla, admin y control remoto en tiempo real necesitás conectar Firebase Realtime Database. Es gratuito y tarda unos 5 minutos.
1 Entrá a console.firebase.google.com → Agregar proyecto → ponele un nombre (ej: fight-night) → seguí los pasos hasta crearlo.
2 Dentro del proyecto → Compilación → Realtime Database → Crear base de datos → elegí cualquier región → Iniciar en modo de prueba.
3 En la pantalla principal del proyecto hacé clic en </> (Agregar app web), ponele nombre, y copiá el objeto firebaseConfig que aparece.
4 Abrí el archivo fight-night.html con cualquier editor de texto, buscá la sección marcada como FIREBASE CONFIG y reemplazá los valores. Guardá y volvé a subir a GitHub Pages.
const FIREBASE_CONFIG = { apiKey: "AIzaSy...", authDomain: "fight-night.firebaseapp.com", databaseURL: "https://fight-night-default-rtdb.firebaseio.com", projectId: "fight-night", storageBucket: "fight-night.firebasestorage.app", messagingSenderId: "123456789", appId: "1:123456789:web:abc123" };
Sin Firebase la app funciona localmente (un solo dispositivo). Los datos no se sincronizan entre pantalla, admin y celular.
